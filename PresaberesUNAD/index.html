Proposito


Esta pagina wed tiene como proposito dar a conocer que son las etiquetas basicas de HTML, sus valores y atributos
dar a conocer como este hipertexto de HTML, es el lenguaje informatico que conforma la mayoria de las paginas wed
y como los usuarios pueden crear, estructurar enlaces implementando etiquetas.



Introducción


Las etiquetas son la estructura básica de HTML, estas se componen de atributos y elementos, HTML es el lenguaje con el que se escriben
la mayoría de las páginas web, este texto puede generar color, fuente de letra, tamaño, imágenes, etc., todo el código HTML se escribe
por medio de etiquetas y atributos, estas etiquetas son palabras o abreviaturas en ingles encerradas en signos “< >” y cada una de las
etiquetas tiene una función, algunas de estas etiquetas tienen atributos que son implementados para completar la información, dado un ejemplo
la etiqueta de imagen “” y su atributo src=”” en la cual se indica ubicación de la imagen que se pretende establecer en el sitio wed,
a su vez encontramos etiquetas que no necesariamente necesitan escribirse con atributos.
